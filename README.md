# The Quiz Room AI App

An AI-powered quiz application built with Python and deployed on Streamlit.

## Overview
The Quiz Room AI App is an interactive quiz platform that leverages artificial intelligence to provide an engaging quiz experience. The application is designed to create, manage, and present quizzes with AI-powered features.

## Features
- AI-powered quiz generation and management
- Interactive quiz interface
- Streamlit-based web application
- Deployed at: https://thequizroom.streamlit.app/

## Technologies Used
- Python
- Streamlit
- Anthropic Claude (for AI functionality)
- Streamlit Cloud for deployment

## Installation and Setup
1. Clone the repository:
   ```
   git clone https://github.com/taha293/The-Quiz-Room-Ai-App.git
   ```

2. Navigate to the project directory:
   ```
   cd The-Quiz-Room-Ai-App
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
   Or if using the project's configuration:
   ```
   uv pip install -r pyproject.toml
   ```

4. Run the application:
   ```
   streamlit run main.py
   ```

## Usage
1. Launch the application
2. Navigate through the quiz interface
3. Interact with the AI-powered quiz features
4. Enjoy the quiz experience

## Project Structure
- `main.py`: Main application file
- `agents_setup.py`: AI agent configuration and setup
- `config.py`: Configuration settings
- `pyproject.toml`: Project dependencies and configuration

## Deployment
The application is deployed on Streamlit Cloud and can be accessed at: https://thequizroom.streamlit.app/
